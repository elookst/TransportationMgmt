Detailed Description

Brief Description - The Carrier class includes all of the attributes and methods involved in tracking, inserting, and updating carrier information.

The Carrier class includes attributes for City, FTLAvailable (number of full loads available), LTLAvailable (number of pallets light load available), FTLRate (full load rate per km), LTLRate (single pallet load rate per km), and ReefCharge (refrigerated trailer)
