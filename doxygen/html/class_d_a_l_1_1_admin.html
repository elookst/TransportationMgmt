Detailed Description

Brief Description - The Admin class includes all of the attributes and methods involved in configuration, reviewing log and alter important data in the database

The Admin class includes methods for accessing the configuration file, review the event log and alter the important information in the database, including rate, carrier and route. This class is called from the main UI and interfaces with the SQL database. The attributes in this class include id, name, and rate which will be used to alter in the database.
